# Gym Management System

![Gym Management System](media/image1.jpeg)

## Overview

The **Gym Management System** is a comprehensive software solution designed to streamline and automate the administrative, operational, and member management tasks within a fitness facility. This system provides an integrated platform that caters to the diverse needs of gym owners, staff, and members, facilitating efficient management of day-to-day operations and enhancing the overall member experience.

## Features

- **Membership Management**: Register, renew, and modify member details.
- **Scheduling and Booking**: Manage class schedules, personal training sessions, and equipment bookings.
- **Attendance Tracking**: Monitor member check-ins and facility usage.
- **Financial Management**: Handle billing, invoicing, and payment processing.
- **Dietary Plan Suggestions**: Generate personalized diet plans based on age and fitness goals.
- **Reporting and Analytics**: Provide insights into member engagement, facility usage, and financial performance.

## Technologies Used

- **Frontend**: Python Tkinter for GUI development.
- **Backend**: JSON for data storage.
- **Programming Language**: Python.

## Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/Gym-Management-System.git

2. Navigate to the Project Directory:
   cd Gym-Management-System

3. Run the Application:
   python gym.py

Usage:
Add Members: Enter member details (ID, name, age, and membership fee) and click "Add Member."

View Members: Click "Show Members" to display all registered members.

Delete Members: Enter the member ID and click "Delete Member."

Diet Plan: Click "Diet Plan" to open a new window where you can generate a personalized diet plan based on age and fitness goals.
